# **App Name**: JU Scholastic

## Core Features:

- Dynamic Semester GPA Calculator: Calculate GPA instantly by adding credit hours and letter grades (A-F) without needing course names.
- Predictive Target Tool: An AI advisor tool that uses reasoning to determine exactly what GPA is needed in future semesters to reach a specific target CGPA based on current academic data.
- Cumulative Semester Tracker: Manage and aggregate multiple semesters to generate a real-time Cumulative Grade Point Average (CGPA).
- Local Semester History: Persist calculation history locally on the device for viewing, editing, or deleting previous records without a network connection.
- Material Dashboard: A centralized view showing current GPA standing, cumulative progress, and history navigation with support for light and dark modes.
- Grade Entry Validation: Real-time verification system that prevents invalid credit hours or impossible grade entries according to Jimma University standards.
- Portal Integration Hub: Visual placeholder for future synchronization with the ju.edu.et student portal to automate transcript data imports.

## Style Guidelines:

- Primary Color: Scholarly Navy (#2E4EBC), chosen to evoke a sense of academic focus and reliability.
- Background Color: Crisp Sky Tint (#F6F7FB), providing a highly legible surface for complex data points.
- Accent Color: Azure Informative (#198BB4), used to highlight calculated results and secondary actions.
- Headline Font: 'Space Grotesk' for a precise, tech-forward, and mathematical aesthetic suitable for data entry.
- Body Font: 'Inter' for its objective and highly readable characteristics across mobile screen sizes.
- Sharp, geometric Material Design icons used sparingly to represent semesters, calculators, and future sync states.
- Vertical scroll-focused grid ensuring easy input for multiple course rows on mobile devices.
- Slick status transitions when toggling between Dark and Light modes to maintain UI continuity.
